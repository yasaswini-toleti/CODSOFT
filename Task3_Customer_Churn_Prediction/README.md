# Customer Churn Prediction

This project predicts whether a customer will leave a bank using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Random Forest

## Dataset Features
- Credit Score
- Geography
- Gender
- Age
- Balance
- Number of Products
- Credit Card Status
- Active Member Status
- Estimated Salary

## Target
Exited
- 0 = Customer Stays
- 1 = Customer Leaves

## Accuracy
Approximately 85%+
