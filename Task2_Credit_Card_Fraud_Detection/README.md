# Credit Card Fraud Detection

This project uses Machine Learning to detect fraudulent credit card transactions.

## Technologies
- Python
- Pandas
- Scikit-learn
- Random Forest

## Dataset
- fraudTrain.csv
- fraudTest.csv

## Features Used
- Transaction Category
- Amount
- Gender
- City Population
- Latitude
- Longitude
- Merchant Latitude
- Merchant Longitude

## Target
is_fraud

- 0 = Legitimate
- 1 = Fraudulent

## Model
Random Forest Classifier

## Accuracy
Approximately 95%+
