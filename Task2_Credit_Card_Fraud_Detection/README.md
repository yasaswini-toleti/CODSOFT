# Credit Card Fraud Detection

This project uses Machine Learning to detect fraudulent credit card transactions.

## Technologies
- Python
- Pandas
- Scikit-learn
- Random Forest

## Dataset

The dataset is too large to upload to GitHub.

Download from:
https://www.kaggle.com/datasets/kartik2112/fraud-detection

Files used:
- fraudTrain.csv
- fraudTest.csv

## Features Used
- Transaction Category
- Amount
- Gender
- City Population
- Latitude
- Longitude
- Merchant Latitude
- Merchant Longitude

## Target
is_fraud

- 0 = Legitimate
- 1 = Fraudulent

## Model
Random Forest Classifier

## Accuracy
Approximately 95%+
